# obsidian-pdf-plus-zh

PDF++ 简体中文汉化版

## 原作者仓库

https://github.com/RyotaUshio/obsidian-pdf-plus

## 安装方法

1. 克隆本仓库到你的 Obsidian 插件目录：
   ```
   D:\obsidian\cangku\仓库名称\.obsidian\plugins
   ```
2. 确保目录中包含 `main.js` 和 `manifest.json` 文件（可能目录下有且仅有他们就可以了，你可以试一下）
3. 重启 Obsidian或在插件设置中启用本插件

## 关于汉化

本仓库为 PDF++ 插件的简体中文本地化版本，基于插件的 i18n 系统实现。
